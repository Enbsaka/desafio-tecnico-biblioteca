# Desafio Técnico - Cadastro de Livros

Projeto desenvolvido como parte de um desafio técnico para vaga de estágio em desenvolvimento.

## Tecnologias utilizadas

- PHP
- MySQL
- Bootstrap

## Banco de dados

Importar o arquivo `database.sql` no MySQL para criar as tabelas necessárias.

## Funcionalidades

- Cadastro de livros
- Listagem de livros
- Edição de livros
- Remoção de livros
